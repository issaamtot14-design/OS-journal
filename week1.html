Week 1 – System Planning & Architecture

This week focused on planning the operating system deployment and defining the system architecture. The aim was to establish a clear baseline for the environment that will be used throughout the remaining weeks (security hardening, testing, optimisation, and auditing).

Key Activities

  • System architecture design (host + VirtualBox + VM)
  • Distribution selection and justification
  • Network configuration planning
  • Initial system information gathering (baseline commands)
